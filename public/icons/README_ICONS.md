# PWAアイコンファイル作成ガイド

## 必要なアイコンサイズ（iOS PWA対応）

以下のサイズのアイコンファイルを作成し、このディレクトリに配置してください：

### 基本アイコン
- `icon-72x72.png` - 72x72ピクセル
- `icon-96x96.png` - 96x96ピクセル
- `icon-128x128.png` - 128x128ピクセル
- `icon-144x144.png` - 144x144ピクセル
- `icon-152x152.png` - 152x152ピクセル
- `icon-192x192.png` - 192x192ピクセル
- `icon-384x384.png` - 384x384ピクセル
- `icon-512x512.png` - 512x512ピクセル

### iOS専用アイコン
- `icon-180x180.png` - 180x180ピクセル（Apple Touch Icon）

## デザインガイドライン

1. **シンプルなデザイン** - JPYCをイメージしたアイコン
2. **背景透明** - PNG形式で背景透明
3. **角丸不要** - iOSが自動的に角丸処理
4. **高解像度** - Retina対応のクリアな画像
5. **統一性** - 全サイズで同じデザイン

## 推奨色
- **プライマリ**: #1f2937 (グレー)
- **アクセント**: #3b82f6 (ブルー)
- **背景**: 透明または白

## 作成ツール推奨
- Adobe Illustrator/Photoshop
- Figma
- Canva
- オンラインPWAアイコンジェネレーター

## 簡易作成方法
1. 512x512の基本アイコンを作成
2. オンラインツールで全サイズを一括生成
   - https://realfavicongenerator.net/
   - https://www.favicon-generator.org/

アイコンファイルが準備できたら、このファイルを削除してください。
